# Тест Selenium для Википедии

Этот проект содержит тесты на Selenium для проверки функциональности поиска в Википедии.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

**Примечание:** ChromeDriver устанавливается автоматически

## Запуск тестов

### Прямой запуск

```
python test_wikipedia_earth.py
```

Будут выполнены оба теста:
- Позитивный тест: проверка содержания кислорода в атмосфере Земли
- Негативный тест: проверка поиска несуществующей статьи

## Описание тестов

### Позитивный тест: `test_wikipedia_earth_oxygen()`

1. Открывает главную страницу русской Википедии
2. Вводит "Земля" в поле поиска
3. Переходит на страницу о Земле
4. Находит информацию о составе атмосферы (div с `style="margin-left:1em"`)
5. Проверяет, что содержание кислорода составляет 20,95%

### Негативный тест: `test_wikipedia_search_nonexistent_article()`

1. Открывает главную страницу русской Википедии
2. Вводит несуществующий запрос в поле поиска
3. Проверяет, что статья не найдена
4. Проверяет наличие индикаторов отсутствия результатов (например, "Создать страницу")

## Требования

- Python 3.7+
- Google Chrome браузер
- Интернет-соединение

## Автоматическая установка драйвера

Проект использует `webdriver-manager` для автоматической установки и управления ChromeDriver. При первом запуске драйвер будет автоматически скачан и установлен в кэш пользователя.
